# PyMES - Small Business HTML Templates

A collection of 10 professional, responsive HTML/CSS templates designed for small businesses (PyMES - Pequeñas y Medianas Empresas). Built with Bootstrap 5 and Tailwind CSS.

## Templates Included

| # | Template | Framework | Description |
|---|----------|-----------|-------------|
| 1 | **Fast Food Restaurant** | Bootstrap | Modern burger restaurant with menu, specials, and online ordering |
| 2 | **Hotel & Tourism** | Tailwind | Luxury resort with room booking, tours, and amenities |
| 3 | **Education** | Bootstrap | Academy/school with programs, faculty, and enrollment |
| 4 | **Healthcare** | Tailwind | Medical clinic with services, doctors, and appointments |
| 5 | **Bakery** | Bootstrap | Artisan bakery with products, custom orders, and gallery |
| 6 | **Ice Cream Store** | Tailwind | Colorful ice cream shop with flavors and locations |
| 7 | **Clothes Store** | Bootstrap | Fashion retail with products, categories, and sales |
| 8 | **Pet Shop** | Tailwind | Pet store with products, services, and care tips |
| 9 | **Small Market** | Bootstrap | Grocery store with categories, deals, and delivery |
| 10 | **Plants Store** | Tailwind | Garden center with plants, supplies, and care guides |

## Features

- Fully responsive design (mobile, tablet, desktop)
- Modern and clean UI/UX
- CDN-based dependencies (no build tools required)
- Cross-browser compatible
- Easy to customize
- Semantic HTML5
- CSS3 animations and transitions
- Bootstrap Icons integration

## Tech Stack

- **Bootstrap 5.3.2** - CSS framework for templates 1, 3, 5, 7, 9
- **Tailwind CSS (CDN)** - Utility-first CSS for templates 2, 4, 6, 8, 10
- **Bootstrap Icons** - Icon library for all templates

## Getting Started

1. Clone this repository:
```bash
git clone https://github.com/krlz-dev/pymes.git
```

2. Navigate to any template folder:
```bash
cd pymes/01-fastfood
```

3. Open `index.html` in your browser

No build process required! All templates use CDN links for frameworks.

## Customization

Each template includes:
- `index.html` - Main HTML file with all sections
- `css/style.css` - Custom styles and overrides
- `images/` - Folder for your images (add your own)

### Adding Your Own Images
Replace the placeholder icons/emojis with actual images by:
1. Adding images to the `images/` folder
2. Updating the `src` attributes in HTML

### Changing Colors
- **Bootstrap templates**: Modify Bootstrap variables or custom CSS
- **Tailwind templates**: Update the `tailwind.config` in the HTML file
